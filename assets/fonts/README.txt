DejaVu Sans (Regular + Bold) — font osadzany w dokumentach PDF operacji magazynowych.

Powód: standardowe fonty PDF (Helvetica/Times) nie mają polskich znaków (ł ą ę ś ć ż ź ń).
DejaVu Sans ma pełne pokrycie i jest wolnodostępny do redystrybucji.

https://dejavu-fonts.github.io/License.html

pliki PDF pozostają małe — repo trzyma jedynie źródłowe TTF.
